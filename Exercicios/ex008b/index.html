Texto Invertido

Para textos invertidos utiliza-se a tag <bdo>, em conjunto também é utilizado o parâmetro dir para indicar uma das duas direções possíveis
rtl = da Direita para Esquerda(right-to-left)
ltr = da Esquerda para a Direita(left-to-right)
